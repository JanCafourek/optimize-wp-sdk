# Optimize WP SDK

The WP optimize package for [Karolína Vyskočilová's](https://kybernaut.cz) WordPress projects. Feel free to use it, if it helps.

## Install

Via Composer

``` bash
$ composer require vyskoczilova/optimize-wp-sdk
```

## Usage

``` php
new Vyskoczilova\OptimizeWP();
```
